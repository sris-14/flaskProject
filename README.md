📝 To-Do App

Hey there 👋
Welcome to my simple To-Do application.

If you're someone who likes keeping tasks organized, this little app helps you add, manage, and remove tasks easily.

✨ What You Can Do

✔️ Add tasks
✔️ Mark tasks as completed
✔️ Delete tasks
✔️ Keep track of what needs to be done

Simple. Fast. No clutter.

🛠 Tech Used

> HTML
> CSS
> JavaScript
> Flask
Render (Deployment)

🚀 Why I Built This

While learning web development, I wanted to build something practical and simple that demonstrates how basic task management works in a web application.

This project helped me understand:

* CRUD operations
* Handling user actions dynamically
* Deploying applications online
